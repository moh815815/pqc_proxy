# نظام وكيل التشفير ما بعد الكمومي
## Arabic Post-Quantum Cryptography (PQC) Web-Proxy Controller

> نظام مفتوح المصدر لإدارة وكيل الويب المقاوم للحواسيب الكمومية، متعدد المستأجرين، مبني بـ Flask + PostgreSQL

---

## هيكل المشروع

```
pqc_proxy/
├── app/
│   ├── __init__.py              # مصنع التطبيق (App Factory)
│   ├── models/
│   │   └── __init__.py          # جميع نماذج قاعدة البيانات (بأسماء عربية)
│   ├── routes/
│   │   ├── auth.py              # مسارات المصادقة والتسجيل
│   │   ├── tenants.py           # إدارة المستأجرين المتعددين
│   │   ├── certificates.py      # إصدار / ترحيل / إلغاء الشهادات
│   │   ├── handshake.py         # آلة حالة المصافحة الكمومية
│   │   ├── compliance.py        # سجل الامتثال والتقارير العربية
│   │   └── algorithms.py        # إدارة خوارزميات PQC
│   └── services/
│       ├── handshake_service.py  # منطق آلة الحالة الكاملة
│       ├── certificate_service.py# دورة حياة الشهادات والترحيل
│       └── compliance_service.py # سجل الامتثال + توليد التقارير
├── config/
│   └── settings.py              # إعدادات البيئات المختلفة
├── migrations/
│   └── 0001_initial.py          # ترحيل قاعدة البيانات الأولي
├── run.py                        # نقطة دخول التطبيق
├── seed.py                       # تهيئة البيانات الأولية
├── requirements.txt
├── Dockerfile
└── docker-compose.yml
```

## التشغيل السريع

### باستخدام Docker
```bash
git clone <repo>
cd pqc_proxy
docker-compose up --build
```

### تشغيل محلي
```bash
# إنشاء بيئة افتراضية
python -m venv venv
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate         # Windows

# تثبيت المتطلبات
pip install -r requirements.txt

# إعداد متغيرات البيئة
export DATABASE_URL="postgresql://pqc_user:pqc_pass@localhost:5432/pqc_proxy_db"
export JWT_SECRET_KEY="مفتاح-سري-آمن"

# ترحيل قاعدة البيانات
flask db upgrade

# زرع البيانات الأولية (خوارزميات NIST + مستأجر تجريبي)
python seed.py

# تشغيل الخادم
python run.py
```

## مرجع API

### المصادقة
```http
POST /api/v1/auth/تسجيل-دخول
Content-Type: application/json

{
  "البريد_الإلكتروني": "admin@pqc-demo.localhost",
  "كلمة_المرور": "Admin@PQC2024!"
}
```

### إصدار شهادة كمومية
```http
POST /api/v1/certificates/إصدار
Authorization: Bearer <token>

{
  "معرف_المستأجر": "uuid-هنا",
  "معرف_الخوارزمية": "uuid-ml-kem-768",
  "الاسم_الشائع": "proxy.company.sa",
  "الدولة": "SA",
  "أيام_الصلاحية": 365
}
```

### ترحيل شهادة
```http
POST /api/v1/certificates/<id>/ترحيل
{
  "معرف_الخوارزمية_الجديدة": "uuid-ml-kem-1024",
  "سبب": "ترقية أمنية إلى المستوى 5"
}
```

### تنفيذ مصافحة كمومية كاملة
```http
POST /api/v1/handshake/مصافحة-كاملة
{
  "معرف_المستأجر": "uuid-هنا",
  "معرف_الخوارزمية": "uuid-ml-kem-768",
  "معرف_الشهادة": "uuid-الشهادة"
}
```

### تقرير الامتثال العربي
```http
GET /api/v1/compliance/تقرير/<معرف_المستأجر>
    ?من_تاريخ=2024-01-01
    &إلى_تاريخ=2024-12-31
```
